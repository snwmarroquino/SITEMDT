# SITEMDT

## Propósito Técnico & Fins de Estudo
Este projeto foi desenvolvido como um laboratório prático para engenharia de renderização e estruturação de layouts. O objetivo principal foi aplicar conceitos avançados de CSS Architecture para garantir máxima performance de carregamento e estabilidade visual (Cumulative Layout Shift - CLS), simulando padrões de alta performance exigidos pelo mercado de tecnologia.

### Core Focus:
- Otimização do caminho crítico de renderização (Critical Rendering Path).
- Estruturação de código limpo e sem redundâncias para redução do tamanho final dos arquivos.
- Responsividade fluida e performance medida via Web Vitals.
